    map-merge-deep 

    @function map-merge-deep($map-old, $map-new, $overwrite: true) { ... }
    View source

    Description

    Perform a deep merge of two maps, optionally overwriting existing values. Note: I have a PR open adding this to Sassy maps

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $map-old

    The map to merge to.

    		 Map — none
    $map-new

    The map to merge into $map-old

    		 Map — none
    $overwrite

    How should the merge deal with existing values? If true, any existing values will be Overwritten. If false, existing values will be preserved. The logic handling overwriting Will fully recurse the map, so merging is based on path rather than node.

    		 Bool true

    Returns

    A new map resulting from the merge of $map-old and $map-new.

    Map
